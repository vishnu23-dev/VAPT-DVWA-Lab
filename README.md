# Vulnerability Assessment and Penetration Testing (VAPT) using DVWA

## Overview
This project demonstrates a Vulnerability Assessment and Penetration Testing (VAPT) process on a deliberately vulnerable web application (DVWA) in a controlled laboratory environment.

## Objectives
- Perform service enumeration
- Analyze HTTP requests
- Identify SQL Injection vulnerabilities
- Demonstrate Reflected XSS vulnerabilities
- Document findings and recommendations

## Tools Used
- Kali Linux
- Nmap
- Burp Suite Community Edition
- DVWA (Damn Vulnerable Web Application)
- Docker

## Methodology

### 1. Reconnaissance
Performed service enumeration using Nmap.

### 2. Traffic Analysis
Captured HTTP requests using Burp Suite.

### 3. Vulnerability Assessment
Tested input fields for common web vulnerabilities.

### 4. Exploitation
Successfully demonstrated:
- SQL Injection
- Reflected Cross-Site Scripting (XSS)

## Findings

### SQL Injection
Severity: High

### Reflected XSS
Severity: Medium
